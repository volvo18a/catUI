# 介绍

----

`cat-UI` 是一款基于 `Vue.js 2.0` , 采取Clarity Design设计语言设计的组件库

## 特性

- 基于 `Vue2.0` 开发的 UI 组件
- 使用 npm + webpack + babel，支持 ES2015, es6
- 友好的 API，灵活使用的组件

## 浏览器支持

- 现代浏览器和 IE9 及以上
