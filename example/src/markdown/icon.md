# Icon 图标

----

语义化的矢量图形, 使用的是ant design 的图标

### 图标实例
注: 根据 type 属性来确定图标类型
:::demo
```html
<cat-icon type="search"></cat-icon>
<cat-icon type="like"></cat-icon>
<cat-icon type="dislike"></cat-icon>
<cat-icon type="copy"></cat-icon>
```
:::
